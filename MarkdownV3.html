Markdown is a lightweight markup language that makes it easy to format plain text. You should use it to create clear, readable documents in places like documentation, forums, and read-me files. When used in GenAI prompts, Markdown helps the model understand the desired structure and presentation, resulting in more organized and accurate responses.

Headings

To create titles and subtitles, organizing content logically.

# Main Title
## Subtitle
### Section

Unordered Lists

To create bulleted lists of items.

- Item 1
- Item 2
- Item 3

Ordered Lists

To create numbered lists of items.

1. First item
2. Second item
3. Third item

Bold Text

Use two asterisks (**) or two underscores (__) around the text to emphasize it with a strong visual weight.

**This text is bold.**
__This is also bold.__

Italic Text

Use a single asterisk (*) or underscore (_) around the text to emphasize text with a softer visual cue.

*This text is italic.*
_This is also italic._

Links

To create a link, put the text you want to be clickable in square brackets ([]) and the URL in parentheses (()).

[Visit Google](https://www.google.com)

Blockquotes

Use to highlight or quote text by adding a greater than sign (>) at the beginning of a line.

> This is a quote.
> It can be multi-line.

Inline Code

To display short snippets of code or commands within a sentence, use a single backtick (`) around the code.

You can use the `print()` function in Python to display output.

Fenced Code Blocks

For a multi-line code block, you use three backticks (```) on a new line before and after the code. You can also specify the programming language for syntax highlighting, which is a neat feature.

```python
print("Hello, World!")
```

Horizontal Rules

Use to visually separate content. You can create one with three or more hyphens (---), asterisks (***), or underscores (___) on a line by themselves.

Here's some content above the line.
***
And here's some content below the line.

Strikethrough

To indicate text that is no longer relevant, wrap the text in two tildes (~~).

~~This is outdated.~~

Images

To embed images: Use an exclamation mark (!) followed by the alt text in brackets and the image URL in parentheses.

![G.I. Joe logo](https://upload.wikimedia.org/wikipedia/en/thumb/e/e3/GIJoeLogo.svg/1200px-GIJoeLogo.svg.png)

Tables

Creating tables in Markdown is a bit more complex, but it's very useful for organizing data. You use hyphens (-) for the header separator and vertical bars (|) to separate columns. The colons (:) in the separator line control the text alignment within the columns.

| Header 1 | Header 2 | Header 3 |
|----------|:--------:|---------:|
| Row 1    | Centered | Right    |
| Row 2    | Left     | 2nd Right|

|--- aligns the text to the left.
|:---:| centers the text.
|---:| aligns the text to the right.
